◆ ResizeFrames() [1/2]

void bvh11::BvhObject::ResizeFrames ( int  num_new_frames)

Change the number of the frames.

When the specified number is larger than the current number, it adds new frames at the end that are uninitialized. When the specified number is smaller than the current frame, it deletes the last frames. When the specified number is the same as the current number, it does not change anything.

num_new_framesNumber of the new frames.
